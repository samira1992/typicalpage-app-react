# typicalpage-app-react

Basic page app  built with React, JavaScript, and CSS.

-----

##### Download:

`$ git clone https://github.com/samira1992/typicalpage-app-react.git`

`$ cd path to the folder`

`$ npm install `


-----------
### Running the code
`npm start`
